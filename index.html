Project Asclepius

An IoT-based distributed health monitoring system that tracks patient vitals — including heart rate, SpO₂, and body temperature — along with ambient conditions such as temperature, humidity, and light. The system connects multiple Raspberry Pi nodes that communicate through a cloud-based Firebase database for asynchronous data synchronization. A web-based dashboard enables real-time visualization and environmental control, such as adjusting lights and ventilation based on live sensor data.

Technologies: Raspberry Pi, Firebase, SQLite, Python, I²C, Wi-Fi, JSON

Concepts: Internet of Things (IoT), distributed systems, sensor integration, cloud communication, real-time monitoring, environmental automation

Autonomous SnowPlough Robot

An autonomous robot designed to clear lightweight cubes within a defined arena while avoiding both static and dynamic obstacles. The system integrates multiple ultrasonic sensors and line-following modules for navigation, and uses DC motors for actuation. Two Arduino boards (Due and Nano Every) communicate via the I²C protocol to coordinate sensor data, movement control, and obstacle avoidance in real time. The modular design enables precise detection, adaptive path planning, and reliable snow-clearing performance under timed testing conditions.

Technologies: Arduino Due, Arduino Nano Every, Ultrasonic Sensors, Line Sensors, DC Motors, I²C Communication

Concepts: Embedded systems, real-time control, sensor fusion, obstacle detection and avoidance, path planning, modular system design

Multi-threaded Elevator Simulator

A real-time, multi-threaded elevator simulation system featuring three primary subsystems — Elevator, Floor, and Scheduler — operating across multiple computers. Technologies: Java, DatagramSocket (UDP), JUnit, UML
